Kanban Board

Organize tasks across stages — from draft to done. Drag, drop, and track progress at a glance.

Human-in-the-Loop

Agents propose plans before writing code. You review, approve, and stay in control of every change.

GitHub Integration

Sync issues, track pull requests, and let agents open PRs directly from your board.

Custom Prompts

Define prompt templates per project. Fine-tune agent behavior for your codebase and workflow.

Real-time Monitoring

Watch agent activity as it happens. Live-streaming logs and status updates via WebSocket.

Multi-Agent

Coordinate multiple AI agents working in parallel. Each with its own permissions and tool access.

How it works

1

Create a task

Describe what you need done, or sync from GitHub issues.

2

Agent plans

The AI agent analyzes your codebase and proposes an implementation plan.

3

You approve

Review the plan. Request changes or give the green light.

4

Agent executes

Watch in real-time as the agent writes code and opens a pull request.

5

Review & merge

Review the PR, run tests, and merge with confidence.

Self-hosted. Your data stays yours.

Run Camelot on your own infrastructure. No data leaves your network. Full control over agent permissions, API keys, and access.
